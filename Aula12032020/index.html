TAGS E SUAS FUNÇÕES

TAGS FUNÇÃO QUEBRA LINHAS?
h1,h2,h3... São utilizadas para indicarem títulos, o tamanho varia de acordo com a tag escolhida Sim
p São utilizadas para criarem um parágrafo Sim
b São utilizadas para deixarem as letras em NEGRITO na versão html 4 Não
br São utilizadas como quebra linhas Sim
strong São utilizadas para deixarem as letras em NEGRITO na versão html 5 Não
i São utilizadas para deixarem as letras em ITÁLICO Não
hr São utilizadas para criarem uma linha Sim
ul São utilizadas para criarem uma lista
ol São utilizadas para criarem uma lista ordenada
table São utilizadas pra criarem uma tabela
thead Esta tag representa o cabeçalho da tabela, geralmente composta por células título
tbody Essa tag representa o corpo da tabela
tfoot Essa tag representa o rodapé da tabela

ID,STYLE E CLASS

TAG FUNÇÃO
id Atribui um id ao elemento (um id deve ser único, ou seja atribuído a um único elemento
style Permite incluir elementos CSS (estilos) dentro da tag
class Atribui uma classe ao elemento (uma classe pode ser utilizada para um ou mais elementos
